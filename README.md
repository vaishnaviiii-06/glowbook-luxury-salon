# glowbook-luxury-salon
An ultra-premium luxury salon website featuring dynamic service searching and responsive appointment booking
# ✦ GlowBook — Bangalore's Finest Beauty Experiences ✦

## ✨ Premium Features
* **Luxury Dark Theme:** Crafted with a sophisticated palette featuring Champagne gold (`#f5ebe0`) and Soft Rose gold (`#d4a373`) accents.
* **Interactive Live Search:** Users can filter luxury services (like Hair Couture, Facials, and Spas) instantly directly from the hero search bar.
* **Smooth Navigation & Motion:** Enhanced with fluid CSS transitions and a responsive sticky navbar.
* **Luxury Booking Engine:** A clean, elegant booking interface with instant reservation success states.

## 🛠️ Tech Stack Used
* **Frontend:** Semantic HTML5, Modern CSS3 (Custom Variables, Flexbox, CSS Grid)
* **Fonts:** Google Fonts (`Cormorant Garamond`, `Cinzel`, `DM Sans`)
* **Interactions:** Vanilla JavaScript (Dynamic input filtering, event handling)

## 📸 Project Showcase
* **Design Philosophy:** Minimalist typography meets premium elite branding.
* **Responsive Layout:** Tailored perfectly for mobile viewports up to large 4K displays.
